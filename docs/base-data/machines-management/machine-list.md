# Machine List

The `Machine List` module is used to maintain machine records, update their setup information, and manage related assignments such as hall and machine state.

## Machine List Overview

![Machine List](/img/basedata/machine-management/machine-list.png)

Go to `Base Data > Machines Management > Machine List` to view all machine records.

### Information Shown on This Page

- **Active**: Shows whether the machine is active.
- **ID**: System-generated machine ID.
- **Machine Name**: Name of the machine.
- **Created On**: Date when the machine was created.
- **Actions**: Options to edit or delete a machine.

### Top Controls

- **Search**: Quickly find a machine by name or ID.
- **New**: Opens the form to create a new machine.
- **All**: Displays all machines.
- **Active**: Displays only active machines.
- **Inactive**: Displays only inactive machines.

## Add a New Machine

![Machine Add](/img/basedata/machine-management/machine-add.png)

Follow these steps to create a new machine:

1. Go to `Base Data > Machines Management > Machine List`.
2. Click the **New** button.
3. Fill in the required machine information in the form.
4. Turn on the **Active** toggle if the machine should be available immediately.
5. Click **Submit** to save the machine.

## Add Machine Hall

![Machine Add Hall](/img/basedata/machine-management/machine-add-machine-hall.png)

If the machine setup includes hall assignment, use this section during creation:

1. Open the machine create form.
2. Go to the **Machine Hall** section.
3. Select the required hall from the available list.
4. Add the hall entry to the machine.
5. Save the machine after finishing the setup.

## Add Machine State

![Machine Add State](/img/basedata/machine-management/machine-add-machine-state.png)

To assign machine states during creation:

1. Open the machine create form.
2. Go to the **Machine State** section.
3. Select the required state from the dropdown or list.
4. Add the state to the machine record.
5. Save the machine after confirming the information.

## Edit a Machine

![Machine Edit](/img/basedata/machine-management/machine-edit.png)

Follow these steps to update an existing machine:

1. Find the machine from the list.
2. Click the **Edit** icon in the relevant row.
3. Update the required machine information.
4. Click **Submit** to save the changes.

## Edit Machine Hall

![Machine Edit Hall](/img/basedata/machine-management/machine-edit-machine-hall.png)

To update a machine's hall information:

1. Open the machine in edit mode.
2. Go to the **Machine Hall** section.
3. Update the assigned hall information.
4. Save the changes.

## Edit Machine Hall Association

![Machine Edit Hall Associate](/img/basedata/machine-management/machine-edit-machine-hall-associate.png)

Use this step when you need to adjust an existing hall association:

1. Open the machine edit screen.
2. Locate the existing hall association entry.
3. Modify the association details as needed.
4. Confirm and save the update.

## Edit Machine Status

![Machine Edit Status](/img/basedata/machine-management/machine-edit-machine-status.png)

To update machine status information:

1. Open the machine in edit mode.
2. Go to the **Machine Status** or **Machine State** section.
3. Change the required status information.
4. Save the machine record.

## Edit Machine Status Association

![Machine Edit Status Associate](/img/basedata/machine-management/machine-edit-machine-status-associate.png)

Use this screen to update an existing status association:

1. Open the machine edit form.
2. Find the current machine status association.
3. Modify the mapped status information.
4. Save the changes.

## Delete a Machine

![Machine Delete](/img/basedata/machine-management/machine-delete.png)

Follow these steps to delete a machine:

1. Locate the machine in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a machine only when you are sure it is no longer needed. If the record may still be required later, keep it inactive instead.
:::
