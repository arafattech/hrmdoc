# Machine Group

The `Machine Group` module is used to maintain group names for machines so similar machines can be organized under a shared category.

## Machine Group List

![Machine Group List](/img/basedata/machine-management/machine-group/machine-group-list.png)

Go to `Base Data > Machines Management > Machine Group` to view the full list of machine groups.

### Information Shown on This Page

- **Active**: Shows whether the machine group is active.
- **ID**: System-generated group ID.
- **Name**: Name of the machine group.
- **Created On**: Date when the group was created.
- **Actions**: Options to edit or delete a machine group.

### Top Controls

- **Search**: Quickly find a machine group by name or ID.
- **New**: Opens the form to create a new machine group.
- **All**: Displays all machine groups.
- **Active**: Displays only active machine groups.
- **Inactive**: Displays only inactive machine groups.

## Add a Machine Group

![Machine Group Add](/img/basedata/machine-management/machine-group/machine-group-add.png)

Follow these steps to create a new machine group:

1. Go to `Base Data > Machines Management > Machine Group`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the group should be available for immediate use.
4. Enter the group name in the **Machine Group Name** field.
5. Click **Submit** to save the record.

### Field Description

- **Active**: Keeps the machine group available in the system.
- **Machine Group Name**: Enter the machine group name. This field is required.

## Edit a Machine Group

![Machine Group Edit](/img/basedata/machine-management/machine-group/machine-group-edit.png)

Follow these steps to update an existing machine group:

1. Find the machine group from the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Machine Group Name** as needed.
4. Click **Submit** to save the changes.

## Delete a Machine Group

![Machine Group Delete](/img/basedata/machine-management/machine-group/machine-group-delete.png)

Follow these steps to delete a machine group:

1. Locate the machine group in the list.
2. Click the **Delete** icon.
3. In the warning popup, click **Delete** to confirm.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a machine group only when you are sure it is no longer required. If it may still be needed, consider marking it inactive instead.
:::
