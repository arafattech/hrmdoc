# Machine State

The `Machine State` module is used to maintain the available machine states such as active, idle, or other operational conditions used across the system.

## Machine State List

![Machine State List](/img/basedata/machine-management/machine-state/machine-state.png)

Go to `Base Data > Machines Management > Machine State` to view all machine state records.

### Information Shown on This Page

- **Active**: Shows whether the machine state is active.
- **ID**: System-generated state ID.
- **Name**: Name of the machine state.
- **Created On**: Date when the state was created.
- **Actions**: Options to edit or delete a machine state.

### Top Controls

- **Search**: Quickly find a machine state by name or ID.
- **New**: Opens the form to create a new machine state.
- **All**: Displays all machine states.
- **Active**: Displays only active machine states.
- **Inactive**: Displays only inactive machine states.

## Add a Machine State

![Machine State Add](/img/basedata/machine-management/machine-state/machine-state-add.png)

Follow these steps to create a new machine state:

1. Go to `Base Data > Machines Management > Machine State`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the state should be available for use.
4. Enter the state name in the **Machine State Name** field.
5. Click **Submit** to save the record.

### Field Description

- **Active**: Keeps the machine state available in the system.
- **Machine State Name**: Enter the machine state name. This field is required.

## Edit a Machine State

![Machine State Edit](/img/basedata/machine-management/machine-state/machine-state-edit.png)

Follow these steps to update an existing machine state:

1. Find the machine state in the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Machine State Name** as needed.
4. Click **Submit** to save the changes.

## Delete a Machine State

![Machine State Delete](/img/basedata/machine-management/machine-state/machine-state-delete.png)

Follow these steps to delete a machine state:

1. Locate the machine state in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a machine state only when you are sure it is no longer required. If it may still be used later, mark it inactive instead.
:::
