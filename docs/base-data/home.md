# Home

![Home](/img/basedata/home.png)

The **Home** page serves as the central hub for the Base Data Management module, providing a high-level overview of system metrics and quick access to essential modules.

## Main Dashboard Cards

The central workspace features "Quick Access" cards that provide snapshots of current data and direct shortcuts:

* **Statistical Cards**: Display real-time record counts for core entities such as **Items**, **Employees**, **Clients**, **Machines**, **Plants**, **Suppliers**, **Storage Locations**, **Operations**, **Tools/Handling Units**, and **Assets**. 
* **Navigation Cards**: Direct action links to administrative features like **Roles & Permissions**, **User Management**, and **Activity** logs.

## Navigation Sidebar

The left-hand sidebar organizes the system's modules into logical sections:

* **Base Data Management**: Contains sub-modules for comprehensive data management including Item Management, Department, Category Management, Supplier, Employee, Client Management, Plant Management, etc. 
* **Settings & Activity**: Found at the bottom for quick access to system configurations and audit history.

## Header Controls

The top navigation bar provides global controls:

* **Sidebar Toggle**: Expand or collapse the left menu for more workspace.
* **System Toggles**: Access the language selector, quick settings, unread notifications, and user profile management.
