# Storage Location

The `Storage Location` module is used to maintain the list of warehouse or storage locations in the system. It allows users to create new locations, update existing ones, and manage their active or inactive status.

## Storage Location List

![Storage Location List](/img/basedata/storage-management/location-list.png)

Go to `Storage Management > Storage Location` to view the full list of storage locations.

### Information Shown on This Page

- **Active**: Shows whether the location is currently active.
- **ID**: System-generated storage location ID.
- **Location Name**: Name of the storage location.
- **Created On**: Date when the location was created.
- **Actions**: Options to edit or delete the location.

### Top Controls

- **Search**: Quickly find a location by name or ID.
- **New**: Opens the form to create a new storage location.
- **All**: Displays all locations.
- **Active**: Displays only active locations.
- **Inactive**: Displays only inactive locations.

## Add a New Storage Location

![Storage Location Add](/img/basedata/storage-management/location-add.png)

Follow these steps to create a new storage location:

1. Go to `Storage Management > Storage Location`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the location should be available for immediate use.
4. Enter the name in the **Location Name** field.
5. Click **Submit** to save the location.

### Field Description

- **Active**: Enable this if the storage location should remain active.
- **Location Name**: Enter the storage location name. This field is required.

## Edit a Storage Location

![Storage Location Edit](/img/basedata/storage-management/location-edit.png)

Follow these steps to update an existing storage location:

1. Find the location in the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Location Name** as needed.
4. Click **Submit** to save the changes.

## Delete a Storage Location

![Storage Location Delete](/img/basedata/storage-management/location-delete.png)

Follow these steps to delete a storage location:

1. Locate the storage location in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a storage location only when you are sure it is no longer required. If it may be needed later, consider marking it inactive instead.
:::
