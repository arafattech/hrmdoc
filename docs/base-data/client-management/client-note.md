# Client Note

The `Client Note` module is used to maintain reusable note labels for client classification. These notes can later be selected from the client form, such as `High priority client` or `Bulk order customer`.

## Client Note List

![Client Note](/img/basedata/client-management/client-note/client-note-list.png)

Go to `Client Management > Client Note` to view all note records.

### Information Shown on This Page

- **Active**: Shows whether the note is currently active.
- **ID**: System-generated note ID.
- **Note Name**: Name of the client note.
- **Created On**: Date when the note was created.
- **Actions**: Options to edit or delete the note.

### Top Controls

- **Search**: Finds a note quickly by name or ID.
- **New**: Opens the note entry form.
- **All**: Displays all notes.
- **Active**: Displays only active notes.
- **Inactive**: Displays only inactive notes.

## Add a New Client Note

![Client Note Add](/img/basedata/client-management/client-note/client-note-add.png)

Follow these steps to create a new client note:

1. Go to `Client Management > Client Note`.
2. Click **New**.
3. Turn on the **Active** toggle if the note should be available for use immediately.
4. Enter the note text in **Note Name**.
5. Click **Submit**.

### Field Description

- **Active**: Keeps the note available in the system.
- **Note Name**: Enter the client note name. This field is required.

## Edit a Client Note

![Client Note Edit](/img/basedata/client-management/client-note/client-note-edit.png)

Follow these steps to update a client note:

1. Find the note in the list.
2. Click the **Edit** icon.
3. Update the **Active** status or **Note Name**.
4. Click **Submit** to save the changes.

## Delete a Client Note

![Client Note Delete](/img/basedata/client-management/client-note/client-note-delete.png)

Follow these steps to delete a client note:

1. Locate the note in the list.
2. Click the **Delete** icon.
3. In the warning dialog, click **Delete** to confirm.
4. Click **Cancel** if you do not want to continue.

:::warning
Do not delete a client note that is already being used in active client records unless you are sure it is no longer needed.
:::
