# Capacities

The `Capacities` module is used to maintain the list of available capacities in the system. It allows users to create, update, search, and manage capacity records for operational or production-related use.

## Capacity List

![Capacities List](/img/basedata/capacities/capacities-list.png)

Go to `Base Data Management > Capacities` to view the full list of capacities.

### Information Shown on This Page

- **Active**: Shows whether the capacity is currently active.
- **ID**: System-generated capacity ID.
- **Capacity Name**: Name of the capacity.
- **Created On**: Date when the capacity was created.
- **Actions**: Options to edit or delete the capacity.

### Top Controls

- **Search**: Quickly find a capacity by name or ID.
- **New**: Opens the form to create a new capacity.
- **All**: Displays all capacities.
- **Active**: Displays only active capacities.
- **Inactive**: Displays only inactive capacities.

## Add a New Capacity

![Capacities Add](/img/basedata/capacities/capacities-add.png)

Follow these steps to create a new capacity:

1. Go to `Base Data Management > Capacities`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the capacity should be available for immediate use.
4. Enter the name in the **Capacity Name** field.
5. Click **Submit** to save the capacity.

### Field Description

- **Active**: Enable this if the capacity should remain active.
- **Capacity Name**: Enter the name of the capacity. This field is required.

## Edit a Capacity

![Capacities Edit](/img/basedata/capacities/capacities-edit.png)

Follow these steps to update an existing capacity:

1. Find the capacity in the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Capacity Name** as needed.
4. Click **Submit** to save the changes.

## Delete a Capacity

![Capacities Delete](/img/basedata/capacities/capacities-delete.png)

Follow these steps to delete a capacity:

1. Locate the capacity in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a capacity only when you are sure it is no longer required. If it may be needed later, consider marking it inactive instead.
:::
