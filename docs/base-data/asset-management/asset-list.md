# Asset

The `Asset List` module allows users to view all assets in the system, create new assets, update existing assets, and remove assets that are no longer needed.

## Asset List

![Asset List](/img/basedata/asset-management/asset/asset-list.png)

Go to `Asset Management > Asset List` to view the full list of assets.

### Information Shown on This Page

- **Active**: Shows whether the asset is currently active.
- **ID**: System-generated asset ID.
- **Asset Name**: Name of the asset.
- **Created On**: Date when the asset was created.
- **Actions**: Options to edit or delete the asset.

### Top Controls

- **Search**: Quickly find an asset by name or ID.
- **New**: Opens the form to create a new asset.
- **All**: Displays all assets.
- **Active**: Displays only active assets.
- **Inactive**: Displays only inactive assets.

## Add a New Asset

![Asset Add](/img/basedata/asset-management/asset/asset-add.png)

Follow these steps to create a new asset:

1. Go to `Asset Management > Asset List`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the asset should be available for use immediately.
4. Enter the asset name in the **Asset Name** field.
5. Click **Submit** to save the asset.

### Field Description

- **Active**: Enable this if the asset should remain active.
- **Asset Name**: Enter the name of the asset. This field is required.

## Edit an Asset

![Asset Edit](/img/basedata/asset-management/asset/asset-edit.png)

Follow these steps to update an existing asset:

1. Find the asset from the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Asset Name** as needed.
4. Click **Submit** to save the changes.

## Delete an Asset

![Asset Delete](/img/basedata/asset-management/asset/asset-delete.png)

Follow these steps to delete an asset:

1. Locate the asset in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to proceed.

:::warning
Delete an asset only when you are sure it is no longer required. Once deleted, recovery may not be easy.
:::
