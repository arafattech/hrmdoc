# Handling Units

The **Handling Units** page is used to create and manage unit names that are used across the system, such as `KG`, `M`, or similar operational units.

## How to Access

1. Open **Base Data Management** from the left menu.
2. Expand **Tools Management**.
3. Click **Handling Units**.

![Handling Units](/img/basedata/tools-management/handling-unit-list.png)

## Handling Units List Page

The list page displays all available units with the following columns:

- **Active**: Indicates whether the unit is active.
- **ID**: System-generated unit ID.
- **Unit Name**: Name of the unit.
- **Created On**: Record creation date.
- **Actions**: Edit and delete options.

Available controls at the top of the page:

- **Search**: Find a specific unit.
- **New**: Add a new handling unit.
- **All**: Show all records.
- **Active**: Show only active units.
- **Inactive**: Show only inactive units.

## Add a New Handling Unit

To create a new handling unit:

1. Open the **Handling Units** page.
2. Click **New**.
3. Set the **Active** toggle if the unit should be usable immediately.
4. Enter the **Unit Name**.
5. Click **Submit** to save the record.
6. Click **Cancel** if you do not want to save.

![Handling Units Add](/img/basedata/tools-management/handling-unit-add.png)

## Edit a Handling Unit

To update an existing unit:

1. Find the required unit from the list.
2. Click the **Edit** icon in the **Actions** column.
3. Modify the **Unit Name** or change the **Active** status.
4. Click **Submit** to save the update.

![Handling Units Edit](/img/basedata/tools-management/handling-unit-edit.png)

## Delete a Handling Unit

To delete a unit:

1. Locate the record in the list.
2. Click the **Delete** icon.
3. A warning message will appear.
4. Click **Delete** to confirm or **Cancel** to close the message.

Delete only when the unit is no longer required.

![Handling Units Delete](/img/basedata/tools-management/handling-unit-delete.png)

## View Inactive Handling Units

To check units that are marked as inactive:

1. Click **Inactive** at the top of the page.
2. The system will show only inactive unit records.
3. If needed, open a record through **Edit** and activate it again.

![Handling Units Inactive](/img/basedata/tools-management/handling-unit-inactive.png)
