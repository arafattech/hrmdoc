# Employee Management

The **Employee Management** module provides a comprehensive platform to handle all employee-related data. It allows HR and management to maintain exact records, including personal information, banking details, salary structures, assets, and important attachments.

## 1. Employee List

To view the list of all configured employees:
1. Go to **Base Data Management > Employee**.
2. This table displays the records of all employees and provides options to add new employees or edit existing ones.

![Employee List](/img/basedata/employee-management/add/employee-list.png)

## 2. Adding a New Employee

To add a new employee, click on the **+ Add** button from the Employee List view. The employee profile creation process is divided into multiple tabs for better organization.

### 2.1 Core Details
Start by entering the **Core** details of the employee. This includes essential workplace information such as their Department, Designation, and Employee ID.

![Employee Add - Core](/img/basedata/employee-management/add/employee-add-core.png)

### 2.2 Personal Details
Next, proceed to the **Personal** tab to input their personal contact information, home address, emergency contacts, and related details.

![Employee Add - Personal](/img/basedata/employee-management/add/employee-add-personal.png)

### 2.3 Bank Details
In the **Bank** tab, you can input the banking information required for payroll distributions.

![Employee Add - Bank](/img/basedata/employee-management/add/employee-add-bank.png)

If you need to add a specific bank account or branch, click on the add buttons within this tab to provide the respective details.

![Employee Add - Bank Add](/img/basedata/employee-management/add/employee-add-bank-add.png)

### 2.4 Salary Details
Navigate to the **Salary** tab to insert their salary structure and base compensation metrics.

![Employee Add - Salary](/img/basedata/employee-management/add/employee-add-salary.png)

### 2.5 Assets
In the **Asset** tab, record any company assets (laptops, phones, id cards, keys) assigned to the employee. 

![Employee Add - Asset](/img/basedata/employee-management/add/employee-add-asset.png)

Click on the internal action buttons to insert new asset assignments for the employee.

![Employee Add - Asset Add](/img/basedata/employee-management/add/employee-add-asset-add.png)

### 2.6 Attachments
Save relevant documents and files like CV, ID copies, or certificates in the **Attachments** tab. 

![Employee Add - Attachments](/img/basedata/employee-management/add/employee-add-attachments.png)

Ensure all information is accurate and click the **Save** button to complete the Employee creation.

---

## 3. Editing an Employee

You can update existing employee records as required as their roles, compensation, or personal details change.
From the Employee List, click the **Edit** action corresponding to the employee.

Like the creation page, the Edit screen consists of several tabs corresponding to different categories of information.

### 3.1 Edit Core Details
Update foundational workplace information.

![Employee Edit - Core](/img/basedata/employee-management/edit/employee-edit-core.png)

### 3.2 Edit Personal Details
Update personal contacts or residential addresses.

![Employee Edit - Personal](/img/basedata/employee-management/edit/employee-edit-presonal.png)

*(Note: Ensure personal records are kept securely updated.)*

### 3.3 Edit Bank Details
Modify the employee bank account information for salary processing.

![Employee Edit - Bank](/img/basedata/employee-management/edit/employee-edit-bank.png)

### 3.4 Edit Salary Details
Revise their compensation structure and salary segments if there is a raise or promotion.

![Employee Edit - Salary](/img/basedata/employee-management/edit/employee-edit-salary.png)

### 3.5 Edit Asset Details
Re-assign or remove assets listed under the employee.

![Employee Edit - Asset](/img/basedata/employee-management/edit/employee-edit-asset.png)

### 3.6 Edit Attachments
Manage the documents associated with this employee's profile by adding new files or removing deprecated ones.

![Employee Edit - Attachments](/img/basedata/employee-management/edit/employee-edit-attachments.png)

### 3.7 View Leave Count
Within the editing views or specific management contexts, you might also be able to review the accumulated leave balances or summary per employee.

![Employee View Leave Count](/img/basedata/employee-management/edit/employee-view-leave-conunt.png)
