# Plant

The `Plant List` module is used to maintain the list of plants in the system. Users can view existing plants, create new plant records, update plant information, and delete plants that are no longer needed.

## Plant List

![Plant List](/img/basedata/plant-management/plant/plant-list.png)

Go to `Plant Management > Plant List` to view all plant records.

### Information Shown on This Page

- **Active**: Shows whether the plant is currently active.
- **ID**: System-generated plant ID.
- **Plant Name**: Name of the plant.
- **Created On**: Date when the plant was created.
- **Actions**: Options to edit or delete the plant.

### Top Controls

- **Search**: Quickly find a plant by name or ID.
- **New**: Opens the form to create a new plant.
- **All**: Displays all plants.
- **Active**: Displays only active plants.
- **Inactive**: Displays only inactive plants.

:::info
Use clear plant names such as `Electrical Plant`, `Utility Plant`, or `Spinning Plant` so users can identify records easily.
:::

## Add a New Plant

![Plant Add](/img/basedata/plant-management/plant/plant-add.png)

Follow these steps to create a new plant:

1. Go to `Plant Management > Plant List`.
2. Click the **New** button.
3. Turn on the **Active** toggle if the plant should be available for use immediately.
4. Enter the plant name in the **Plant Name** field.
5. Click **Submit** to save the record.

### Field Description

- **Active**: Enable this if the plant should remain active.
- **Plant Name**: Enter the name of the plant. This field is required.

## Edit a Plant

![Plant Edit](/img/basedata/plant-management/plant/plant-edit.png)

Follow these steps to update an existing plant:

1. Find the plant in the list.
2. Click the **Edit** icon in the relevant row.
3. Update the **Active** status or **Plant Name** as needed.
4. Click **Submit** to save the changes.

## Delete a Plant

![Plant Delete](/img/basedata/plant-management/plant/plant-delete.png)

Follow these steps to delete a plant:

1. Locate the plant in the list.
2. Click the **Delete** icon.
3. In the confirmation popup, click **Delete**.
4. Click **Cancel** if you do not want to continue.

:::warning
Delete a plant only when you are sure it is no longer required. If it may still be needed later, consider marking it inactive instead.
:::
