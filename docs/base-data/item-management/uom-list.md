# UOM List

This section provides details and working steps for managing the Unit of Measurement (UOM) list.

![UOM List](/img/basedata/uom-list/uom-list.png)

**How it Works (Working Steps):**
1. Navigate to **Base Data Management** > **Item Management** > **UOM List**.
2. The page presents a comprehensive table displaying all available UOMs, detailing their Active status, ID, Name, and Created On date.
3. Use the **Search** field to locate a specific UOM by name or ID.
4. Toggle between **All**, **Active**, and **Inactive** tabs to filter the main view.
5. Use the **Edit** and **Delete** icons inside the Actions column to manage existing records.

# UOM Add

Documentation for creating a new Unit of Measurement (UOM).

![UOM Add](/img/basedata/uom-list/uom-add-list.png)

**How it Works (Working Steps):**
1. Click the **New** button situated at the top right of the UOM List page.
2. A **New** creation modal will securely prompt on the screen.
3. Use the **Active** toggle switch to set the default state of the UOM.
4. Provide the appropriate measurement name in the mandatory **Name** input field.
5. Complete the process by clicking the **Submit** button, or hit **Cancel** to dismiss the form.
