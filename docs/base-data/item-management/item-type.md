# Item Type

This section provides details and working steps for managing Item Types.

![Item Type](/img/basedata/item-type/item-type.png)

**How it Works (Working Steps):**
1. Navigate to **Base Data Management** > **Item Management** > **Item Type**.
2. The page displays a list of all existing Item Types with columns for Active status, ID, Name, and Created On date.
3. Use the **Search** bar to quickly find specific item types.
4. Filter the records using the **All**, **Active**, and **Inactive** buttons at the top right.
5. In the **Actions** column, click the **Edit** icon to modify an item type, or the **Delete** icon to remove it.

# Add Item Type

Documentation for creating a new Item Type.

![Add Item Type](/img/basedata/item-type/item-add-type.png)

**How it Works (Working Steps):**
1. Click on the **New** button at the top right corner of the Item Type page.
2. A **New** pop-up modal will appear.
3. Toggle the **Active** switch to specify if the new item type should be active (turned on by default).
4. Enter the required descriptive name in the **Type Name** field.
5. Click the **Submit** button to save the new item type, or click **Cancel** to close the modal without saving.
