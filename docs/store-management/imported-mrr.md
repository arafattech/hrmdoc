# Imported MRR

This module allows you to handle Material Receipt Reports (MRR) for imported goods.

## Add Imported MRR
To create a new Imported MRR:
1. Navigate to the **Imported MRR** section.
2. Click the **Add New** button.
3. Fill out the relevant supplier, shipment, and item details before saving.

![Add Imported MRR](/img/store/imported-mrr/imported-mrr-add.png)

## View Imported MRR
To view a specific imported MRR:
1. From the list, click the view icon on an MRR item.
2. Review the detailed receipt properties.

![View Imported MRR](/img/store/imported-mrr/imported-mrr-view.png)
