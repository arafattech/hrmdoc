# Payroll History

The **Payroll History** page gives HR administrators a complete record of all past payroll batches. It is split into two panels — the **Payroll History list** on the left, and the **Payroll (employee breakdown)** panel on the right.

![Payroll History](/img/hr/payroll-management/payroll-history/payroll-history.png)

## Payroll History List (Left Panel)

The left panel shows a list of all payroll batches that have been generated. Each row represents one payroll run with the following columns:

| Column             | Description                                               |
|--------------------|-----------------------------------------------------------|
| **Active**         | Indicates if the payroll batch is active (✓)             |
| **Start Date**     | The beginning of the payroll period                       |
| **End Date**       | The end of the payroll period                             |
| **Total Employees**| Number of employees included in that payroll run          |
| **Total Salary**   | Combined total salary paid in that batch                  |
| **Payment Method** | The method used to make the payment (if applicable)       |

**Steps to view a payroll batch:**
1. Navigate to **Payroll Management → Payroll History** from the left sidebar.
2. The list of all past payroll batches will be shown on the left.
3. Click on any row to select a payroll batch.
4. The right panel will update to show the **individual employee breakdown** for that batch.

## Employee Payroll Breakdown (Right Panel)

When you click on a payroll batch from the left panel, the right panel shows the list of employees included in that batch.

Each employee row shows:

| Column             | Description                                       |
|--------------------|---------------------------------------------------|
| **Active**         | Whether the employee was active during this period|
| **Employee ID**    | Unique ID of the employee                         |
| **Employee Name**  | Full name of the employee                         |
| **Basic Salary**   | The base salary processed for this employee       |
| **Payment Method** | The payment method used for this employee         |
| **Net Salary**     | The final amount paid (after deductions)          |

## Filter by Total Employees

You can filter the payroll history list by the number of employees in a batch.

![Payroll History Filter](/img/hr/payroll-management/payroll-history/payroll-history-filter.png)

**Steps to apply the filter:**
1. Click the **filter icon (▽)** next to the **Total Employees** column header.
2. A filter option will appear. Enter or select the number of employees to filter by.
3. The list will update to show only payroll batches matching the selected employee count.

> For example, filtering by **3** will show only payroll runs where exactly 3 employees were included.

## Full-Screen Payroll View

You can expand the right panel to full screen to get a better view of the employee payroll breakdown.

![Payroll History Full Screen](/img/hr/payroll-management/payroll-history/payroll-history-payroll-full-screen.png)

**Steps to enter full-screen mode:**
1. Click the **expand (›)** arrow button on the divider between the left and right panels.
2. The right panel (employee payroll table) will expand to full width.
3. Click the arrow again **( ‹ )** to return to the split-panel view.

The full-screen view shows the same columns — **Active, Employee ID, Employee Name, Basic Salary, Payment Method, and Net Salary** — in a wider, easier-to-read format.

## Search

Both the left panel (Payroll History list) and the right panel (Employee list) have a **Search** bar. Use the search box to quickly find:
- A specific payroll period (left panel)
- A specific employee within a payroll batch (right panel)
