# Payroll

The **Payroll** page shows a complete list of all processed payroll records. From here, HR administrators can view payroll summaries, check individual payroll details, and download payslips.

![Payroll](/img/hr/payroll-management/payroll/payroll-list.png)

## Summary Cards

At the top of the Payroll page, you will see four summary cards:

| Card                  | Description                                           |
|-----------------------|-------------------------------------------------------|
| **Current Employees** | Total number of active employees (e.g., 621)          |
| **Last Month Employees** | Employees from the previous payroll month           |
| **Total Salary**      | Combined gross salary of all processed payrolls       |
| **Total Outstanding** | Total amount still pending/unpaid                     |
| **Total Requests**    | Total number of payroll requests processed            |

> These cards give a quick overview of payroll status at a glance.

## Employee Payroll List

Below the summary cards, the **Employee Payroll** table lists all payroll records with the following columns:

- **Active** — Whether the employee is currently active (✓).
- **First Name / Last Name** — Employee's name.
- **Payable Days** — Number of days the employee is eligible to be paid.
- **Payment Status** — Shows **PROCESSED** (blue badge) once payroll has been generated.
- **Payroll Time** — The date range for the payroll period (e.g., 05/03/2026 – 05/04/2026).
- **Process Date** — The actual date the payroll was processed.
- **Basic Salary, Total Allowance, Night Shift, Attendance Bonus, Total Deduction, Net Salary** — Full breakdown of pay components.
- **Action** — Contains a **View** (👁) button and a **Download** (⬇) button.

## View Payroll Details

To see the full payroll details for a specific employee:

1. Find the employee in the payroll list.
2. Click the **👁 (eye icon)** under the **Action** column.
3. A **Details** popup will appear showing:

![Payroll View](/img/hr/payroll-management/payroll/payroll-view.png)

| Field               | Description                                      |
|---------------------|--------------------------------------------------|
| Employee ID         | Unique employee identifier                       |
| Employee Name       | Full name of the employee                        |
| Basic Salary        | Base salary amount                               |
| Total Bonus         | Combined bonus (attendance + night shift, etc.)  |
| Allowance           | Total allowances added                           |
| Payment Status      | Current status (e.g., *processed*)               |
| Total Attendance    | Number of attendance days recorded               |
| Process Date        | Date payroll was finalized                       |
| Payable Days        | Total days the employee is to be paid for        |
| Payroll Time        | Payroll period (start date – end date)           |
| Total Night         | Total night shift hours/days worked              |
| Night Shift Bonus   | Bonus earned for night shifts                    |
| Attendance Bonus    | Bonus for attendance (green)                     |
| Total Deductions    | Sum of all deductions (shown in red)             |
| **Net Salary**      | Final take-home amount after deductions          |
| HR Notes            | Any notes added by HR for this record            |

4. Click **Cancel** to close the popup.

## Download Payslip

To download an individual payslip:

1. Click the **⬇ (download icon)** in the **Action** column next to the employee's row.
2. The payslip will be downloaded automatically.

![Payroll Download](/img/hr/payroll-management/payroll/payroll-download.png)

You can also use the **download icon** at the top-right of the table (next to the search bar) to **download all payroll records** in bulk.

## Filter by Date Range

To view payroll for a specific date period:

![Payroll Date Range](/img/hr/payroll-management/payroll/payroll-date-range-filter.png)

**Steps:**
1. Click the **date range field** at the top-right (e.g., *05/03/2026 – 06/04/2026*).
2. A calendar picker will open. Select your **Start Date** and **End Date**.
3. Click the **Go** button to apply the filter.
4. The payroll list will refresh to show only payrolls within the selected date range.
5. Click **Reset** to clear the filter and return to the default view.

## Search

Use the **Search** bar (top-right of the table) to quickly find a specific employee's payroll record by typing their name or employee ID.

## Payroll Reset

The **Reset** button allows you to clear any applied date range filter and return the payroll list to its default view.

![Payroll Reset](/img/hr/payroll-management/payroll/payroll-reset.png)

**Steps:**
1. After applying a date range filter using the **Go** button, click the **Reset** button (located to the right of the **Go** button).
2. The date range will reset to the default payroll period.
3. The **Employee Payroll** list will reload and show all available payroll records.

> Use the Reset button whenever you want to remove any date filter and go back to the full payroll list.
