# Asset

The **Asset** section allows employees to track assets provided to them by the company.

## Asset List

To view your allocated assets, follow these steps:

1. Navigate to the **Employee Dashboard** from the left sidebar.
2. Select the **Asset** tab located in the top navigation menu.
3. You will see an **Asset** table displaying all items assigned to you.

**Features of the Asset List:**
- **Search bar:** Quickly find a specific asset using the search field at the top right.
- **Table Information:** The table displays detailed columns including:
  - `S/N`
  - `Asset` (e.g., Computer)
  - `Issue Date`
  - `Return Date`
  - `Remarks`
  - `Note`
