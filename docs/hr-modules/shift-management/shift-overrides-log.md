# Shift Overrides Log

The **Shift Overrides Log** tracks all manual changes made to system-calculated rosters. Any manual **Swap**, **Replacement**, or **Extra Entry** on the Roster page is recorded here for audit and payroll purposes.

![Shift Overrides Log Overview](/img/hr/shift-management/shift-overrides-log/shift-overides-log-overview.png)

## Log Breakdown

The table provides detailed tracking of each override:

- **ID**: The unique ID for the override log entry.
- **Shift**: The specific shift affected (e.g., Day, Night, Morning).
- **Swap / Replacement Employee Name**:
  - Displays the name of the original employee if swapped.
  - Shows the new staff member assigned.
- **Roster Date**: The date the shift override applies to.
- **Action Date**: The date the override was entered into the system.
- **Changed Type**: Categorizes the type of modification:
  - **EXTRA**: Additional staff added beyond the standard roster.
  - **SWAP**: Two employees exchanged shifts.
  - **REPLACEMENT**: One employee was replaced by another.

## Using Search

Use the **Search** bar at the top right to filter by:
1. **Employee Name**: Find logs for specific staff.
2. **Override Status**: Search for "EXTRA" or "SWAP" keywords.
3. **Shift ID**: Reference specific entries using the ID.

:::info
The data in this log eventually flows into the Payroll Analysis to ensure correct calculations for overtime, attendance, and shift differentials.
:::
