# Daily Attendance

The **Daily Attendance** page provides a real-time card-based view of all employees' attendance status for a selected date and shift. Each employee is shown as a card displaying their name, Employee ID, shift, check-in time, check-out time, and total working hours. You can instantly see who is **Present** (green) and who is **Absent** (red).

---

## Daily Attendance List

![Daily Attendance List](/img/hr/employee-management/daily-attendance/daily-attendance.png)

The main attendance view displays all employees for the currently selected shift and date. The top toolbar contains the following controls:

- **Shift Dropdown** — Select a shift (e.g., P/A) to filter employees by shift.
- **Date Picker** — Select a specific date to view attendance records for that day.
- **Department Dropdown** — Filter employees by department.
- **Search Bar** — Search for a specific employee by ID or name.
- **Reset** — Clear all filters and return to the default view.
- **All / Present / Absent** — Quick filter buttons to show all employees, only present employees, or only absent employees.

Each employee card shows:
- **Employee Name** and **ID**
- **Shift** assigned to the employee
- **Check-In Time** — The time the employee checked in for the day
- **Check-Out Time** — The time the employee checked out (shown as `--` if not yet checked out)
- **Working Hours** — Total hours worked so far (e.g., 5h 3m) or `0h` if absent

---

## View Present Employees

![Daily Attendance Present List](/img/hr/employee-management/daily-attendance/daily-attendance-present.png)

To view only the employees who are **Present** today, click the **Present** button in the top right corner. The list will filter to show only the employees who have checked in, displaying their check-in time and total working hours.

---

## View Absent Employees

![Daily Attendance Absent List](/img/hr/employee-management/daily-attendance/daily-attendance-absent.png)

To view only the employees who are **Absent** today, click the **Absent** button in the top right corner. The list will filter to show only absent employees, each showing `0h` working hours and no check-in or check-out time.

---

## Filter by Department

![Daily Attendance Filter Department](/img/hr/employee-management/daily-attendance/daily-attendance-filter.png)

To filter attendance records by a specific department:

1. Click the **Department** dropdown in the top toolbar.
2. Select the department you want to view (e.g., **Electrical & Utility**).
3. The attendance cards will update to show only employees belonging to that department.

You can combine this with the **Shift** dropdown, **Date** picker, and the **All / Present / Absent** buttons for more refined filtering.

---

## Search by Employee Name or ID

![Daily Attendance Filter Name](/img/hr/employee-management/daily-attendance/daily-attendance-name-filter.png)

To search for a specific employee:

1. Type the employee's **name** or **ID** into the **Search by ID or Name** field in the top toolbar.
2. The attendance cards will instantly filter to show only the matching employee(s).

You can combine the name search with the department filter for even more specific results.

---

## Reset Filters

![Daily Attendance Reset](/img/hr/employee-management/daily-attendance/daily-attendance-reset.png)

To clear all applied filters and return to the full attendance view:

1. Click the **Reset** button in the top toolbar.
2. All filter selections (Shift, Date, Department, Search) will be cleared, and the page will display all employees for the current day.
