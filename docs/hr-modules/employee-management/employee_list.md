# Employee List

The **Employee List** page displays all employees in the system. It shows key details such as Employee ID, First Name, Last Name, Designation, Email, Department, Supervisor, Contact Number, Total Salary, and Created On date. You can filter employees by **All**, **Active**, or **Inactive** status using the filter buttons at the top right.

![Employee List](/img/hr/employee-management/employee/employee-list.png)

---
## Selecting Employees from the List

You can select one or multiple employees using the checkboxes on the left side of the list. As you select employees, an **Action** button will appear at the top, along with a counter showing how many employees have been selected.

![Employee List Select Employee](/img/hr/employee-management/employee/employee-list-select.png)

## Selected Employee Actions (Active / Inactive)

Once you have selected an employee, click the **Action** button to open the actions dialog. This menu allows you to quickly adjust the status of the selected employee(s):
- If the selected employee is currently **Inactive**, you can choose to make them **Active**.
- If the selected employee is currently **Active**, you can choose to make them **Inactive**.

This is a fast way to update employee statuses without needing to edit their individual profiles.

![Employee Selected Action Click to Active Or Inactive](/img/hr/employee-management/employee/employee-list-select-action-click.png)

## Add New Employee

To add a new employee, click the **New** button on the top right of the Employee List page. A dialog box will appear with multiple tabs: **Core Data**, **Personal Data**, **Salary**, **Bank**, **Assets**, and **Attachments**.

### Step 1 — Core Data

![Employee Add Core Data](/img/hr/employee-management/employee/add/employee-add-core.png)

Fill in the following fields in the **Core Data** tab:

- **Active** — Toggle to set the employee as active or inactive.
- **Shift** — Toggle to assign a shift to the employee.
- **Employee ID** — Auto-generated unique ID for the employee.
- **First Name** *(required)* — Enter the employee's first name.
- **Last Name** — Enter the employee's last name.
- **Email** — Enter the employee's email address.
- **Contact Number** *(required)* — Enter the employee's contact number.
- **Department** — Select the employee's department from the dropdown.
- **Designation** — Select the employee's designation.
- **Employee Type** — Select the type of employee (e.g., Staff, Operator).
- **Supervisor** — Select the employee's supervisor.
- **Employee Section** — Select the section the employee belongs to.

Click **Submit** to save the core data or move to the next tab.

---

### Step 2 — Personal Data

![Employee Add Personal Data](/img/hr/employee-management/employee/add/employee-add-personal.png)

Fill in the following fields in the **Personal Data** tab:

- **Gender** — Select the employee's gender.
- **Religion** — Select the employee's religion.
- **NID/Birth Certificate** — Enter the National ID or Birth Certificate number.
- **Referred By** — Enter the name of the person who referred this employee.
- **Emergency Contact Person Name** — Enter the name of an emergency contact.
- **Emergency Contact Person Number** — Enter the phone number of the emergency contact.
- **Emergency Contact Person Relation** — Enter the relationship of the emergency contact.
- **Blood Group** — Select the employee's blood group.
- **Date of Joining** — Select the employee's joining date from the calendar.
- **Date of Birth** — Select the employee's date of birth.
- **Present Address** *(required)* — Enter the employee's current address.
- **Same as Present Address** — Check this box to auto-fill the permanent address with the present address.
- **Permanent Address** — Enter the employee's permanent address.

---

### Step 3 — Salary

![Employee Add Salary Data](/img/hr/employee-management/employee/add/employee-add-salary.png)

Fill in the salary details in the **Salary** tab:

- **Basic** *(required)* — Enter the basic salary amount.
- **House Rent** — Enter the house rent allowance amount.
- **Medical Allowance** — Enter the medical allowance amount.
- **Transport Allowance** — Enter the transport allowance amount.
- **Other Allowance** — Enter any other allowance amount.
- **Total Salary** *(required)* — The total salary is calculated automatically based on all the above fields.

---

### Step 4 — Bank

![Employee Add Bank List Data](/img/hr/employee-management/employee/add/employee-add-bank-list.png)

The **Bank** tab shows a list of bank accounts linked to the employee. Click the **New** button inside this tab to add a new bank account.

#### Add Bank Account

![Employee Add Bank Add](/img/hr/employee-management/employee/add/employee-add-bank-add.png)

Fill in the following fields to add a bank account:

- **Primary** — Toggle to mark this account as the primary account.
- **Bank Name** — Enter the name of the bank.
- **Branch Name** — Enter the branch name.
- **Account Number** — Enter the bank account number.
- **Account Holder Name** — Enter the name of the account holder.
- **Swift Code** — Enter the bank's Swift/BIC code.

Click **Submit** to save the bank account.

---

### Step 5 — Assets

![Employee Add Asset List Data](/img/hr/employee-management/employee/add/employee-add-asset-list.png)

The **Assets** tab displays the list of assets assigned to the employee. Click the **New** button inside this tab to assign a new asset.

#### Add Asset

![Employee Add Asset Add Data](/img/hr/employee-management/employee/add/employee-add-asset-add.png)

Fill in the following fields to assign an asset:

- **Asset** — Select an asset from the dropdown list.
- **Issued Date** — Select the date the asset was issued.
- **Return Date** — Select the expected return date of the asset.
- **Remarks** — Enter any relevant remarks about the asset.
- **Note** — Enter any additional notes.

Click **Submit** to save the asset assignment.

---

### Step 6 — Attachments

![Employee Add Attachment Data](/img/hr/employee-management/employee/add/employee-add-attachment-upload.png)

The **Attachments** tab allows you to upload documents or photos related to the employee (e.g., profile picture, ID documents).

- Click the **Upload** button to select and upload a file from your computer.
- Uploaded files will be listed with their file name, upload date, and file size.
- You can also **Download** existing attachments or remove them using the **×** button.

Click **Submit** to save all information and create the employee record.

## Edit Employee

To edit an existing employee, click the **Edit (pencil) icon** in the Action column of the Employee List.

### Edit Core Data

![Employee Edit Core Data](/img/hr/employee-management/employee/edit/employee-edit-core.png)

The **Core Data** tab opens by default when editing an employee. All fields that were entered during creation are pre-filled and can be updated:

- **Active / Shift** toggles
- **Employee ID**, **First Name**, **Last Name**
- **Email**, **Contact Number**
- **Department**, **Designation**, **Employee Type**, **Supervisor**, **Employee Section**, **Employee Shift**

The employee's profile photo (uploaded as an attachment) is displayed in the top-right corner.

Click **Submit** to save your changes.

---

### Edit Personal Data

![Employee Edit Personal Data](/img/hr/employee-management/employee/edit/employee-edit-personal.png)

Update the employee's personal details in the **Personal Data** tab, including Gender, Religion, NID/Birth Certificate, Emergency Contact details, Blood Group, Date of Joining, Date of Birth, and Address information.

---

### Edit Salary Data

![Employee Edit Salary Data](/img/hr/employee-management/employee/edit/employee-edit-salary.png)

Update the employee's salary breakdown in the **Salary** tab — Basic, House Rent, Medical Allowance, Transport Allowance, Other Allowance, and Total Salary.

---

### Edit Bank List Data

![Employee Edit Bank List Data](/img/hr/employee-management/employee/add/employee-add-bank-list.png)

View and manage the employee's linked bank accounts. You can add a new bank account or edit/delete existing ones directly from this tab.

---

### Edit Bank Add

![Employee Edit Bank Add](/img/hr/employee-management/employee/edit/employee-add-bank-list.png)

Add a new bank account to the employee's profile by clicking the **New** button inside the Bank tab and filling in the required fields.

---

### Asset List Data

![Employee Add Asset List Data](/img/hr/employee-management/employee/add/employee-add-asset-list.png)

View the list of assets currently assigned to the employee. You can add new assets or edit and remove existing ones.

---

### Asset Add Data

![Employee Add Asset Add Data](/img/hr/employee-management/employee/add/employee-add-asset-add.png)

Assign a new asset to the employee by clicking the **New** button inside the Assets tab and filling in the required fields.

---

### Leave Summary

![Employee Leave Count](/img/hr/employee-management/employee/edit/employee-edit-leave.png)

The **Leave** tab shows a read-only summary of the employee's leave status:

- **Total Leave** — The total number of leave days taken by the employee.
- **Leave Balance (Year)** — The remaining leave balance available for the current year.

---

### Edit Attachments

![Employee Add Attachment Data](/img/hr/employee-management/employee/edit/employee-edit-attachment.png)

Upload, download, or remove attachments for the employee. Previously uploaded files are listed with their file name, upload date, and size. Use the **Upload** button to add new files.

---

## Activate an Employee

![Employee Active](/img/hr/employee-management/employee/employee-active.png)

To **activate** an employee, open the Edit dialog and go to the **Core Data** tab. Toggle the **Active** switch to the **ON** (blue) position and click **Submit**. The employee will appear with a checkmark (✓) in the Active column of the Employee List.

---

## Deactivate an Employee

![Employee InActive](/img/hr/employee-management/employee/employee-inactive.png)

To **deactivate** an employee, open the Edit dialog and go to the **Core Data** tab. Toggle the **Active** switch to the **OFF** (grey) position and click **Submit**. The employee's active status will be removed from the main list.

---

## View Inactive Employees

![Employee Inactive List](/img/hr/employee-management/employee/employee-inactive-list.png)

To view all inactive employees, click the **Inactive** filter button at the top right of the Employee List page. Inactive employees are displayed with an **×** symbol in the Active column instead of a checkmark.

---

## Delete an Employee

![Employee Delete](/img/hr/employee-management/employee/add/employee-delete.png)

To delete an employee, click the **Delete (trash) icon** in the Action column of the Employee List. A warning dialog will appear asking:

> **Are you sure you want to delete?**

- Click **Delete** to confirm and permanently remove the employee record.
- Click **Cancel** to go back without deleting.
