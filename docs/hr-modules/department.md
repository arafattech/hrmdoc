# Department

The **Department** module is used to manage the organization structure by creating and maintaining departments for employees.

![Department List](/img/hr/employee-management/employee_department/hrm_employee_department.png)

## Department List

The main table shows the available departments with:

- A row selection checkbox
- **Active** status
- **Custom ID**
- **Name**
- **Created On**
- **Actions** for edit and delete

Use the controls at the top right to:

- Search for a department
- Create a new department with **New**
- Filter the list by **All**, **Active**, or **Inactive**

## Create Department

Click **New** to open the department creation dialog.

The form includes:

- **Active** toggle
- **Department Name**

Enter the department name and click **Submit** to save it.

## Edit Department

Use the **Edit** icon in the Actions column to update an existing department.

The edit dialog opens with the current department data prefilled so you can adjust the name or active state before saving.

## Delete Department

Use the **Delete** icon in the Actions column to remove a department.

You will be prompted to confirm the deletion before it is applied.

:::note
Departments are typically used as a base organizational unit for employee assignment, filtering, and reporting.
:::
