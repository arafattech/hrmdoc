# Mobile Application

The **Mobile Application** section covers the HR functions available on mobile devices for employees and supervisors.

## What It Includes

- **Employee self-service** for viewing profile, attendance, leave, and payroll data
- **Leave actions** for requesting and tracking leave from a phone
- **Attendance and calendar views** for quick daily access
- **Supervisor workflows** for approvals and related HR tasks

## Purpose

This section is used to document the mobile-facing HR experience so users can quickly find the features available in the app.
