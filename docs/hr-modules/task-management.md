# Task Management

The **Task Management** module is used to create, assign, track, and complete work items across the organization. It helps HR and supervisors keep tasks visible, organized, and accountable.

## Task List

The task list provides a quick view of all tasks in the system.

Typical columns include:

- **Task ID**
- **Title**
- **Assignee**
- **Priority**
- **Due Date**
- **Status**
- **Created On**
- **Actions** for edit, view, and delete

Use the list controls to:

- Search for a task by title or assignee
- Filter tasks by status, priority, or date
- Switch between active and completed tasks

## Create Task

Click **New** to create a task.

The task form typically includes:

- **Task Title** *(required)*
- **Description**
- **Assignee**
- **Priority**
- **Due Date**
- **Status**

After entering the details, click **Submit** to save the task.

## Assign and Track Tasks

Once a task is created, it can be assigned to a specific user or employee. The assignee can then update the task as work progresses.

Common status values include:

- **Pending**
- **In Progress**
- **Completed**
- **Cancelled**

Use the status field to keep the task list accurate and to track progress over time.

## Edit Task

Use the **Edit** action to update task details such as title, assignee, priority, due date, or status.

## Delete Task

Use the **Delete** action to remove a task that is no longer needed.
