# Calendar Management

The **Calendar Management** module allows HR to configure holidays, off-days, and special events for individuals, departments, or the entire organization. It is divided into a calendar view on the left and a management panel on the right.

![Calendar Management](/img/hr/leave-management/calender-management/calender-list.png)

## Calendar Overview

The left panel displays the monthly calendar for a selected month and year.

- **Legend**: Colored dots indicate different types of events:
  - 🟠 **Public Holiday**
  - 🟢 **Govt Holiday**
  - 🔵 **Special Event**
  - 🟡 **Industrial Holiday**
  - 🟣 **Religious Holiday**

- **Filter Controls**: Select the desired **Month** and **Year** at the top-left of the calendar.

## Individual and Corporate Management

Use the management panel on the right to set holidays and off-days.

![Calendar Filter Individual And Institute](/img/hr/leave-management/calender-management/calender-filter-leave-type.png)

**Selecting Scope**: Ensure you select the correct category from the dropdown:
- **Individual**: For specific employees.
- **Industrial**: For company-wide holidays.

### Managing Off Days (Individual)

To set recurring weekly off-days for a department or individual:

![Calender Employee Filter](/img/hr/leave-management/calender-management/calender-filter.png)

**Steps to select a target:**
1. Select **Individual** in the first dropdown.
2. Select a **Department** (e.g., **Hr**).
3. Select a specific **Employee** (e.g., **10004 - Arif**).

![Calender Filter Individual And Institute](/img/hr/leave-management/calender-management/calender-filter-select-day.png)

**Steps to apply off-days:**
1. Under **SELECT WEEKEND DAYS**, check the boxes for the days needed (e.g., **Sun**, **Mon**, **Tue**).
2. Select an **APPLICATION SCOPE**:
   - **Entire Year**: Applies the selected weekends for the whole year.
   - **Specific Month**: Applies only for a single month.
3. Click the **Apply** button.
4. Review the **CURRENT HOLIDAYS** list that appears below.
5. Click the **Save** button at the bottom-right.

![Calender Select Save Button](/img/hr/leave-management/calender-management/calender-select-save-btn.png)

### Managing Industrial Holidays

To set company-wide holidays, select **Industrial** from the scope dropdown.

![Calender Filter Industrial Event Type](/img/hr/leave-management/calender-management/calender-filter-industrial.png)

**Steps to add holidays:**
1. Select **Industrial** in the first dropdown.
2. Click **+ ADD HOLIDAY OR SPECIAL DAYS**.
3. Select a **Start Date** and **End Date**.
4. Choose the **Event Type** (e.g., Public Holiday, Govt Holiday).
5. Click **Add Selected Dates**.
6. The dates will appear in the **CURRENT HOLIDAYS** list. You can remove any by clicking the **Delete (🗑)** icon.

![Calender Select Date](/img/hr/leave-management/calender-management/calender-select-date.png)

7. Click the **Save** button to finalize.

:::tip
Always check the **Legend** below the calendar to ensure the correct colors are applied after saving.
:::
