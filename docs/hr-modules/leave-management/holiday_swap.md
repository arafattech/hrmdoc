# Holiday Swap

The **Holiday Swap** module allows employees to exchange one off-day or holiday for another as per company policy. HR can manage and review these swap requests.

![Holiday Swap](/img/hr/leave-management/holiday-swap/holiday-swap-list.png)

## Holiday Swap Logs

The main table lists all previous and current holiday swap requests:

| Column          | Description                                              |
|-----------------|----------------------------------------------------------|
| **S/N**         | Serial number.                                           |
| **Employee Name**| Full name of the employee requesting the swap.           |
| **Employee ID** | Unique Identifier for the employee.                      |
| **Original Date**| The date originally scheduled as an off-day or holiday.  |
| **Transfer To** | The target date to swap with.                           |
| **Swap Type**   | Category of the swap (e.g., WEEKEND).                   |
| **Created On**  | Date the swap request was submitted.                    |

## Creating a New Swap Request

To add a new swap request for an employee:

1. Click the **New** button at the top-right of the table.

![Holiday Swap Employee](/img/hr/leave-management/holiday-swap/holiday-swap-employee.png)

A **Swap** popup will appear:

**Steps to submit a swap:**
1. Select the **Swap Type** (e.g., **Weekend**).
2. Select the **Employee** (e.g., **10001 - Abdur Rahman**).
3. Under **Available Weekend**, the employee's existing off-days will list automatically.
4. Select an **Original Date** (e.g., **Apr 11, 2026**).
5. Choose the **Transfer To** date (e.g., **Apr 15, 2026**).
6. Click the **Swap Icon (⇄)** on the right to link the dates.
7. Click the **Submit** or **Save** button (if available) to record the swap.
8. To exit without saving, click **Cancel**.

:::note
Only available off-days or weekends for the selected employee will be displayed for selection.
:::
