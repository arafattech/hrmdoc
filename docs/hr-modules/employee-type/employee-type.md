# Employee Type

The **Employee Type** module enables you to categorize your workforce into groups such as Operator, Officer, Staff, Worker, etc. This categorization simplifies payroll rules, leave allocations, and attendance policies.

## Viewing Employee Types

The main list displays all registered employee types:

- **Active**: A checkmark indicates the type is active.
- **Name**: The name of the employee type.
- **Created On**: When the type was added.
- **Actions**: Options to **Edit** or **Delete**.

You can filter by **All**, **Active**, and **Inactive** or use the search bar to find a specific type.

![Employee Type List](/img/hr/employee-type/employee-type-list.png)

## Adding an Employee Type

To add a new employee categorization:

1. Click the **New** button.
2. Ensure the **Active** toggle is enabled to use this type immediately.
3. Enter a new **Type Name** (e.g., "Contractor").
4. Click **Submit**.

![Add Employee Type](/img/hr/employee-type/employee-type-add.png)

## Editing an Employee Type

To rename or deactivate an employee type:

1. Click the **Edit** icon.
2. Make your necessary changes to the name or active status.
3. Click **Submit**.

![Edit Employee Type](/img/hr/employee-type/employee-type-edit.png)

## Managing Inactive Employee Types

Types that are not currently assigned to any active staff can be marked as inactive. 

1. Go to the **Inactive** tab.
2. If you need to make an inactive type active again, click the **Edit** icon and switch the **Active** toggle back on.

![Inactive Employee Type](/img/hr/employee-type/employee-type-inactive.png)

## Deleting an Employee Type

To completely remove an employee type from the system:

1. Click the **Delete** icon next to the record.
2. A confirmation prompt will ask: *"Are you sure you want to delete?"*
3. Select **Delete** to cleanly process the removal.

![Delete Employee Type](/img/hr/employee-type/employee-type-delete.png)
