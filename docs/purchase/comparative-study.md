# Comparative Study

This module allows you to compare different quotations from various suppliers side-by-side to make optimal purchasing decisions.

## Comparative List
To view comparative study lists:
1. Navigate to the **Comparative Study** module.
2. The main page displays a list of comparative analyses prepared from submitted RFQs.

![Comparative List](/img/purchase/comparative-study/comparative-list.png)

## Create Purchase Order from Comparative Study
To proceed with a decision and generate a Purchase Order (PO):
1. Open a specific comparative study to review the items and quoted prices.
2. Select the optimal quotation or supplier, then locate the option to create a PO.
3. Confirm the details in the modal or prompt to initialize the Purchase Order based on those agreed values.

![Create PO](/img/purchase/comparative-study/comparative-create-po.png)
