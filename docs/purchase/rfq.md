# RFQ

The Request for Quotation (RFQ) module helps you create and manage RFQs sent to suppliers or external vendors.

## RFQ List
To view all requests for quotation:
1. Go to the **RFQ** module from the side menu.
2. The main view displays a tabulated list of all existing RFQs, showing essential details such as code, status, and supplier info.

![RFQ List](/img/purchase/rfq/rfq-list.png)

## View RFQ Details
To view the details of a specific RFQ:
1. Click the view icon corresponding to an item in the RFQ list.
2. The detailed view provides an item-by-item breakdown and other related parameters of the request.

![RFQ List View](/img/purchase/rfq/rfq-list-view.png)

## Add a New RFQ
To create a new RFQ:
1. Click the **Add New** button on the top right of the RFQ list page.
2. Fill out the required details such as supplier, requested items, dates, and instructions.
3. Save or submit the form to officially generate the RFQ record.

![Add RFQ](/img/purchase/rfq/rfq-add.png)
