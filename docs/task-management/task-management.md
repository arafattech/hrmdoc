# Task Management

The **Task Management** module is used to organize work in a dedicated workspace, create projects, plan work, and keep task progress visible across the team.

![Task Workspace](/img/hr/task_managment/task_management_dashbaord.png)

## Task Workspace

The main dashboard is called **Task Workspace**. It gives a quick overview of current work and exposes the main entry points for task activity.

From this screen you can:

- Create a new project with **New Project**
- Open the planning area with **Plan**
- Review summary cards for:
  - **Total Projects**
  - **Total Tasks**
  - **In Progress Tasks**
  - **Completed**
- Jump to quick actions such as **Task List** and **Kanban Board**

The left sidebar provides direct navigation to:

- **Task Dashboard**
- **My Task**
- **Task List**
- **Kanban Board**
- **Project Management**

## Planning

Click **Plan** from the dashboard to open the planning area.

![Planning List](/img/hr/task_managment/planning_create.png)

The planning screen shows a table of existing plans with:

- **Plan Title**
- **Status**
- **Start Date**
- **Created On**
- **Action** icons for edit and delete

Use the search bar to find a plan quickly, or click **New Plan** to create a new one.

## Create Project

Click **New Project** from the dashboard to start a new project.

![Create Project](/img/hr/task_managment/start_with_creating_porject.png)

The project form includes:

- **Project Name** *(required)*
- **Description**

Enter the project details and click **Submit** to create the project.

## Task Operations

The dashboard links tasks to the broader planning and project workflow.

Common actions in this module include:

- Opening the **Kanban Board** for visual task tracking
- Reviewing **My Task** for personal assignments
- Managing projects from **Project Management**
- Updating plans and removing obsolete entries
