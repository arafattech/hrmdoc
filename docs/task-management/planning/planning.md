# Planning

Click **Plan** from the dashboard to open the planning list.

![Planning List](/img/hr/task_managment/planning_create.png)

The planning screen shows:

- **Plan Title**
- **Status**
- **Start Date**
- **Created On**
- **Action** icons for edit and delete

Use the search bar to find a plan quickly, or click **New Plan** to create a new one.
