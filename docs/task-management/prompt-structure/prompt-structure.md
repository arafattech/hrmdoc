# Prompt Structure

The **Prompt Structures** page is available under **Project Management** in the sidebar. It is used to manage reusable prompt templates.

![Prompt Structure List](/img/hr/task_managment/promt-structure-list.png)

The list view shows a table of saved templates with:

- **ID**
- **Title**
- **Created On**
- **Action** icons for edit and delete

At the top of the page you can:

- Search existing templates
- Click **New** to add a prompt structure
- Open the settings icon for page actions

![Prompt Structure Create](/img/hr/task_managment/promt-structure-create-new.png)

The create form includes:

- **Prompt Structure Title** *(required)*
- **Description**

Use **Submit** to save the new structure.

![Prompt Structure Edit](/img/hr/task_managment/promt-structure-edit.png)

The edit modal lets you:

- Change the **title**
- Update the **description**
- Save the revised prompt structure with **Submit**
