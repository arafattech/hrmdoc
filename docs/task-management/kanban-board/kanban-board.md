# Kanban Board

The **Kanban Board** provides a visual workflow for tracking tasks across status columns.

![Kanban Board](/img/hr/task_managment/kanban-board.png)

The board is organized into:

- **Backlog**
- **In Progress**
- **Review**
- **Done**

At the top of the board you can:

- Select a **Project**
- Select a **Current Plan**
- Switch between **Grid** and **Calendar**

The filtered view shows the board after the project and plan are selected.

![Kanban Board Filter](/img/hr/task_managment/kanban-board-filter.png)
