# Task Workspace

The task workspace is the main entry point for the module.

![Task Workspace](/img/hr/task_managment/task_management_dashbaord.png)

From this screen you can:

- Create a new project with **New Project**
- Open the planning area with **Plan**
- Review summary cards for total projects, total tasks, in progress tasks, and completed tasks
- Jump to quick actions such as **Task List** and **Kanban Board**
