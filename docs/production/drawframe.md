# Drawframe

The `Drawframe` page is used to monitor drawframe machine performance from the production dashboard. It allows users to check machine condition quickly and understand whether the drawframe machine is producing or waiting.

## Machine Drawframe List

![drawframe-machine-list](/img/production/drawframe/drawframe-machine-list.png)

Go to `Production > Drawframe` to open this machine list.

### What Users Can See

- **Summary cards** for efficiency and machine counts.
- **Machine cards** for the selected drawframe section.
- **Status**, **Temperature**, **Load**, and **Uptime**.
- **Card**, **Compact**, and **Full Screen** controls for different monitoring styles.

### How to Use This Page

1. Open `Production > Drawframe`.
2. Review the machine counter at the top of the page.
3. Check the machine status badge.
4. Compare temperature, load, and uptime values.
5. Switch to `Full Screen` when you want a clearer machine display.

### Drawframe Status Meaning

- **Active**: The machine is currently producing.
- **Idle**: The machine is on but not giving production.
- **Inactive**: The machine is off.
