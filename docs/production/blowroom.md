# Blowroom

The `Blowroom` page is used to monitor all blowroom machines from the production dashboard. It helps users quickly see which blowroom machines are active, idle, or inactive and review key operating values such as temperature, load, and uptime.

## Blowroom Machine List

![blowroom-machine-list](/img/production/blowroom/blowroom-machines-list.png)

Go to `Production > Blowroom` to open this page.

### What Users Can See

- **Top Summary Cards**: Overall efficiency and machine status count for the selected section.
- **Machine Monitoring Dashboard**: List of blowroom machines in the selected view.
- **Status Badge**: Shows whether a machine is `Active`, `Idle`, or `Inactive`.
- **Temperature**: Current temperature reading.
- **Load**: Current machine load.
- **Uptime**: Running time shown for the machine.
- **View Switcher**: `Card`, `Compact`, and `Full Screen`.

### How to Use This Page

1. Open `Production > Blowroom`.
2. Check the summary cards at the top.
3. Review each machine status.
4. Use `Compact` view to see more machines at once.
5. Use `Card` view when you want easier reading for each machine.
6. Use `Full Screen` for detailed visual monitoring on a large screen.

## Blowroom Machine Details

![blowroom-machine-details](/img/production/blowroom/blowroom-machines-card.png)

The blowroom page also supports an expanded machine card view. This makes it easier to compare a small number of blowroom machines.

### Blowroom Monitoring Notes

- `Active` means the machine is currently producing.
- `Idle` means the machine is on but production is not happening.
- `Inactive` means the machine is off.

:::tip
Use this page at the beginning of a shift to verify whether all blowroom machines are running as expected.
:::
