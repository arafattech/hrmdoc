# Carding

The `Carding` page is used to monitor carding machines in real time. From this page, users can understand whether the carding machine is producing, waiting, or stopped and can review the main machine readings shown on the dashboard.

## Machine Carding List

![carding-machine-list](/img/production/carding/carding-machine-list.png)

Go to `Production > Carding` to view carding machine status.

### What Users Can See

- **Overall Efficiency** for the selected machine group.
- **Active / Idle / Inactive / Under Maintenance** counters.
- **Machine Monitoring Dashboard** with machine cards.
- **Temperature**, **Load**, and **Uptime** for each carding machine.
- **Card**, **Compact**, and **Full Screen** view options.

### How to Use This Page

1. Open `Production > Carding`.
2. Review the top counters to understand the current machine status.
3. Check the machine card to confirm whether the carding machine is producing.
4. Use `Card` view for better readability.
5. Use `Full Screen` if you need a larger individual machine view.

### Carding Status Meaning

- **Active**: The carding machine is in production.
- **Idle**: The carding machine is on but not producing.
- **Inactive**: The carding machine is off.
