# Utility

The `Utility` page is used to monitor utility-related machines from the same production dashboard. It helps users observe supporting machines that affect the production environment.

## Utility Machine list

![utility-machine-list](/img/production/utility/utility-machine-list.png)

Go to `Production > Utility` to open this page.

### What Users Can See

- **Overall Efficiency** and machine status counters.
- **Utility machine cards** with status and key readings.
- **Temperature**, **Load**, and **Uptime** data on each machine card.
- **Card**, **Compact**, and **Full Screen** view options.

### How to Use This Page

1. Open `Production > Utility`.
2. Check the summary counters at the top.
3. Review the utility machine cards one by one.
4. Check whether each machine is active, idle, or inactive.
5. Use `Card` or `Full Screen` view for easier reading.

### Utility Status Meaning

- **Active**: The machine is running and supporting production.
- **Idle**: The machine is on but not currently producing or supplying output.
- **Inactive**: The machine is off.
