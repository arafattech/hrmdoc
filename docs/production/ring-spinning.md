# Ring Spinning

The `Ring Spinning` page is used to monitor ring spinning machines from the production dashboard. It helps users identify which machines are running, which are idle, and which are stopped.

## Machine Ring Spinning List

![ring-spinning-machine-list](/img/production/ring-spinning/ring-spinning-machine-list.png)

Go to `Production > Ring Spinning` to view this page.

### What Users Can See

- **Overall Efficiency** and machine status counters.
- **Machine Monitoring Dashboard** for ring spinning machines.
- **Machine Status Badge** such as `Inactive` or `Active`.
- **Temperature**, **Load**, and **Uptime** on each card.
- **Card**, **Compact**, and **Full Screen** view options.

### How to Use This Page

1. Open `Production > Ring Spinning`.
2. Review the top counters.
3. Check whether the machine is active, idle, or inactive.
4. Use the machine card values to identify abnormal conditions.
5. Use `Full Screen` when you need a larger machine view for monitoring.

### Ring Spinning Status Meaning

- **Active**: The machine is in production.
- **Idle**: The machine is on, but it is not producing.
- **Inactive**: The machine is off.
