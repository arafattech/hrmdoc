# Autoconer

The `Autoconer` page is used to monitor autoconer machines in the production dashboard. It gives a simple real-time view of machine status and operating values.

## Machine Autoconer List

![autoconer-machine-list](/img/production/autoconer/autoconer-machine-list.png)

Go to `Production > Autoconer` to view this page.

### What Users Can See

- **Overall Efficiency** and machine count summary.
- **Active / Idle / Inactive / Under Maintenance** counters.
- **Machine Monitoring Dashboard** for autoconer machines.
- **Temperature**, **Load**, and **Uptime** for each machine.
- **Card**, **Compact**, and **Full Screen** view controls.

### How to Use This Page

1. Open `Production > Autoconer`.
2. Review the summary cards to understand the current machine condition.
3. Check the autoconer machine card.
4. Use `Card` view for standard monitoring.
5. Use `Compact` or `Full Screen` if you need a different monitoring layout.

### Autoconer Status Meaning

- **Active**: The machine is currently in production.
- **Idle**: The machine is on but not producing.
- **Inactive**: The machine is off.
